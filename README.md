# 🎓 Graduation Students Employment Analysis
### 毕业生就业去向数据清洗与可视化分析

本项目通过对毕业生规模、就业单位行业分布及地域分布等多维度数据的挖掘，旨在分析近年来的就业市场趋势。项目结合了信息管理专业的数据处理逻辑与 UI/UX 视觉传达设计。

## 📌 项目亮点
* **多维数据清洗**：处理涵盖毕业生人数、行业、地域等维度的异构数据。
* **动态交互分析**：利用 `Plotly` 实现可缩放、可悬停查看的交互式图表。
* **信息架构优化**：将复杂的统计结果重构为逻辑清晰、视觉统一的分析海报。

## 📂 仓库结构
* `graduation_students_analysis.ipynb`: 核心分析脚本。包含从数据导入、清洗、描述性统计到生成交互式图表的全过程。
* `analysis-infographic.jpg`: 最终分析成果的可视化海报，展示了数据洞察与美学设计的结合。
* `requirements.txt`: 项目依赖列表，确保环境一键配置。
* **数据源文件**:
    * `毕业人数.xlsx`
    * `单位地域.xlsx`
    * `单位行业.xlsx`

## 🛠️ 技术栈
* **数据处理**: Python (`Pandas`, `Numpy`)
* **交互式可视化**: `Plotly` (`graph_objects`, `express`)
* **静态绘图**: `Matplotlib`, `Seaborn`
* **设计辅助**: Adobe Photoshop (用于成果海报的信息排版与视觉优化)

## 🚀 快速开始

### 1. 环境准备
建议在 Python 3.8+ 环境下运行。克隆仓库后，安装必要的依赖库：
```bash
pip install -r requirements.txt
```

### 2. 运行分析
启动 Jupyter Notebook 并打开 `graduation_students_analysis.ipynb`：
```bash
jupyter notebook
```
